## 🛠️ Твои задачи (Sprint: Frontend & UI)

### 1. Архитектура UI (Glassmorphism + Dark Mode)
Создай премиальный интерфейс в `src/app/globals.css`.
- Фон: Темно-синий/глубокий черный с неоновыми градиентами (subtle radial gradients).
- Стили: Используй утилиты Tailwind для стекла (`backdrop-blur-md`, `bg-white/5`, `border-white/10`).

### 2. Главный Layout (`src/app/layout.tsx` и страницы)
Сверстай 3-панельную структуру SaaS-платформы:
1. **Левый Сайдар (Навигация):**
   - Профиль пользователя и баланс (в рублях/токенах).
   - Ссылки: "Операционный набор", "Advisory Board", "Менторская группа", "Мои пространства".
2. **Центральная панель (Chat / Workspace):**
   - Основной интерфейс взаимодействия, похожий на Manus/Claude.
   - Поле ввода внизу с кнопкой прикрепления файлов (скрепка).
   - Область для вывода сообщений агентов (поддержка Markdown-реплик).
3. **Правая панель (Status / Context):**
   - Виджет статуса: показ, какой агент (Claude, Gemini, Qwen) сейчас "думает".
   - Список прикрепленных документов в текущем пространстве.

### 3. Экран авторизации (SSO UI)
В папке `src/app/login/page.tsx` сверстай красивый экран входа.
Бэкенд-роут для NextAuth уже готов (`/api/auth/signin`). Тебе нужно только сделать красивые кнопки "Войти через Google", "Войти через Yandex" и форму "Email / Пароль".

### 4. Страница Тарифов (Pricing Page / Billing)
В папке `src/app/pricing/page.tsx` сверстай 3 карточки тарифов (Starter: 2900₽, Professional: 9900₽, Business: 29900₽).
Бэкенд для оплаты **уже готов**. Тебе нужно, чтобы кнопка "Купить" делала POST запрос на `/api/checkout` с телом `{ "planId": "starter/professional/business" }`. Бэкенд вернет `confirmationUrl`, на который нужно перенаправить пользователя.
